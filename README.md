# Hunter's Archive | HxH Fan Site

A modern, interactive fan site dedicated to the Hunter x Hunter universe. Built with React, TypeScript, and Tailwind CSS, featuring stunning animations and an immersive user experience.

## ✨ Features

- **Hero Section** - Eye-catching landing page with animated elements
- **Hunter License Generator** - Interactive component to create your own Hunter License
- **Nen Types Explorer** - Detailed breakdown of all six Nen categories
- **Character Roster** - Showcase of iconic characters from the series
- **Arc Gallery** - Visual journey through major story arcs
- **Nen Quiz** - Interactive quiz to discover your Nen type
- **Responsive Design** - Fully optimized for desktop and mobile devices

## 🛠️ Tech Stack

- **React 19.2.3** - UI library
- **TypeScript 5.9.3** - Type safety
- **Vite 7.2.4** - Build tool and dev server
- **Tailwind CSS 4.1.17** - Utility-first styling
- **Framer Motion 12.38.0** - Smooth animations
- **Lucide React** - Beautiful icons

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd hunter-x-hunter-fan-site
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
